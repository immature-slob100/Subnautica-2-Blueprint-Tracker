## Overview

Subnautica 2 Blueprint Tracker is a companion utility designed to help players keep track of discovered blueprints, unlocked technologies, crafting progression, and important resources.

Whether you are exploring new biomes or completing your technology tree, Blueprint Tracker keeps your progression organized.

---

## Features

### Blueprint Tracking

* Track discovered blueprints
* Monitor unlock progress
* View technology completion

### Resource Database

* Resource locations
* Material requirements
* Crafting dependencies

### Exploration Progress

* Track discovered biomes
* Monitor exploration goals
* Review progression milestones

### Statistics

* Blueprint completion rate
* Technology progress
* Exploration percentage
* Resource collection overview

## Installation

1. Download the latest release.
2. Extract the ZIP archive.
3. Launch Subnautica2BlueprintTracker.exe.
4. Create a profile.
5. Start tracking your progress.
