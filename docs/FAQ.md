# Frequently Asked Questions

## Does this modify game files?

No. Blueprint Tracker is a companion application and does not modify game files.

## Can I use multiple profiles?

Yes. Multiple explorer profiles are supported.

## Does it affect performance?

No. The application runs independently from the game.

## Can I track blueprint progress manually?

Yes. Blueprint entries can be managed manually.

## Is an internet connection required?

No. Core functionality works offline.

## Is this an official Subnautica 2 tool?

No. This is a community-made companion project.
